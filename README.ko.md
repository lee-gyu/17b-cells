# 17b cells

엑셀은 행 1,048,576개 열 16,384개로 구성되어 총 171,7986,9184개의 셀을 가집니다.
이 프로젝트는 웹 브라우저에서 엑셀과 동일한 셀 개수를 출력할 수 있도록 기술적인 도전을 구현하였습니다.

## 기본 구현 사항

1. 스프레드시트의 각 좌표를 표시하는 UI가 맨 좌측와 맨 상단에 고정되어져 보여집니다.
2. 데이터를 내보내기(Export), 불러오기(Import)가 가능해야 합니다.
3. 최대한 효율적인 자료구조를 사용해서 메모리 및 저장 공간을 최적화해야 합니다.

## 최적화 아이디어

1. 기본 값을 기반으로 모든 것을 초기화합니다.
   - 예를 들어, 기본적인 행 높이 값은 24px이며, 행 크기를 변경하지 않은 경우 이 값은 별도의 메모리로 생성하지 않습니다.

### State

`State`는 현재 스프레드시트 내의 상태를 관리하는 작업을 담당합니다.

- 현재 상태를 스냅샷
- 다음 상태로 넘어갈 때, 어떤 점만 변경되었는지 변경점만 기록 (History)

### View

`View`는 등록된 Schema와 State를 통해 어떤 UI를 화면에 보여주어야 하는지 결정하는 작업을 담당합니다.

#### Virtualizer

현재 높이/너비에 맞는 셀만 보여줄 영역만 계산하는 모듈
